ESP-Moonlight 是乐鑫推出的基于 ESP32 开发的月球灯示例项目，内含产品开发的完整步骤、最佳做法，可实现驱动 LED 灯、局域网内使用小程序端控制灯的亮度和色彩等参数。本项目以快速开发为特点，硬件结构简单，代码架构清晰完善，方便功能扩展，可用于教育领域。

<img src="docs/_static/moonlight2.jpg" alt="moonlight" width=800 />



月球灯有以下功能：

- 板载自动下载电路
- 电池充电功能
- 支持手机进行配网
- 支持手机控制
- 支持触碰控制
- 支持OTA升级
- 支持语音控制
